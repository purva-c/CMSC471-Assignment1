About This Visualization

Dataset

This dataset contains daily U.S. weather measurements from 2017, provided by the NOAA Daily Global Historical Climatology Network (GHCN). It includes readings from weather stations across all U.S. states and territories, covering temperature, precipitation, snowfall, and wind data to reveal seasonal and geographic patterns across stations.

Design Rationale

We use a scatter plot to reveal relationships between weather variables across stations and time. The month slider enables temporal exploration (a key dimension of weather data). Coloring by state (optional) allows geographic grouping. Tooltips provide station-level detail on demand when you hover over a data point.

Interactions

  • Month slider — filters data by month, enabling temporal exploration (dynamic query)
  • Axis dropdowns — change X, Y, and size variables to explore different variable relationships
  • Tooltips — hover over any point for station name, state, date, and variable values
  • Color by State toggle — switch between a temperature-based color scale and state-based coloring
